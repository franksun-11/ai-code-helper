# 项目学习路线

---

## 学习路线概览

本学习路线包含 4 个阶段，从新手入门到技术进阶，帮助你系统地提升编程能力和项目经验。

- **阶段 1**：新手入门 - 适合初学者，系统学习项目开发流程
- **阶段 2**：真实业务实战 - 完整业务场景，提升工程能力
- **阶段 3**：快速补充技术栈 - 针对性学习特定技术
- **阶段 4**：技术进阶 - 深入架构设计和系统优化

---

## 阶段 1 - 新手入门

> **适用人群**：编程新手、应届生、转行人员  
> **学习建议**：建议按照顺序全部都做，根据自己学习的方向只做前端或后端，觉得简单就快速跳过

### 1. 用户中心项目

**适合人群**：后端和前端新手入门

**学习目标**：
- 系统学习完整的项目开发流程
- 掌握项目上线方法
- 建立全栈开发概念

**核心技术**：
- 前端：HTML/CSS/JavaScript 基础
- 后端：Java + Spring Boot
- 数据库：MySQL
- 版本控制：Git

---

### 2. 伙伴匹配系统

**适合人群**：有一定基础的初学者

**学习目标**：
- 巩固开发流程
- 学习后端核心技术

**核心技术**：
- Redis 缓存
- 事务处理
- 并发编程
- 大数据推荐思想

---

### 3. AI 编程助手（2025年最新）⭐

**适合人群**：想要入门 AI 应用开发的新手

**学习目标**：
- 实战 LangChain4j 框架的主流用法和特性

**核心技术**：
- **对话记忆**：实现上下文对话
- **结构化输出**：格式化 AI 响应
- **AI Service**：服务化 AI 能力
- **RAG**（检索增强生成）：知识库问答
- **工具调用**：Function Calling
- **MCP**（Model Context Protocol）
- **SSE**（Server-Sent Events）：实时流式输出

---

### 4. AI 程序员技术练兵场（2025年最新）⭐

**适合人群**：想要系统学习 AI 应用开发的同学

**学习目标**：
- Java + Vue 全栈 AI 应用开发
- 企业级 AI 应用开发流程和技巧

**核心技术**：
- LangChain4j AI 框架
- 结构化输出
- 提示词工程（Prompt Engineering）
- 企业级开发规范

---

## 阶段 2 - 真实业务实战

> **适用人群**：有一定项目经验的开发者  
> **学习建议**：选择 2-3 个项目，学完后要求能够独立完成 1 个类似项目的开发和上线

### 1. AI 零代码应用生成平台（2025年最新）🔥

**项目特点**：以 AI 开发 + 后端架构为核心的微服务全栈项目，对标大厂

**核心技术**：

#### AI 相关
- **LangChain4j AI 智能体**
- **LangGraph4j 工作流**
- Vibe Coding

#### 微服务架构
- Spring Cloud
- Dubbo
- Nacos
- Sentinel

#### 系统优化
- 多维度系统优化
- 响应式编程
- 企业级监控体系

#### 其他技术
- 多种设计模式
- COS 对象存储
- Selenium 自动化
- Nginx

**学习收获**：
- 前沿技术实践
- 提升编程技能
- 系统架构设计能力

---

### 2. 智能协同云图库（2025年最新）🔥

**项目特点**：基于 Vue 3 + Spring Boot + COS + WebSocket 的企业级智能协同云图库平台

**核心业务场景**：
- 文件存管
- 内容检索
- 权限控制
- 实时协同

**核心技术**：
- **MySQL 分库分表**
- **Redis + Caffeine 多级缓存**
- **COS 对象存储**
- **Sa-Token 权限控制**
- **DDD 领域驱动设计**
- **WebSocket 实时通讯**
- **JUC（Java 并发编程）**
- **Disruptor**（高性能队列）
- **AI 绘图大模型**
- **设计模式**

---

### 3. AI 答题应用平台

**项目特点**：深入业务场景，跨端开发 + 后端架构

**核心技术**：

#### 前端
- React 跨端小程序开发
- Vue3 AI 应用网站开发

#### 后端
- 分库分表
- 分布式锁
- 缓存设计
- 幂等设计
- 设计模式
- **RxJava 响应式编程**
- **SSE 实时推送**
- 线程池隔离

**学习收获**：
- 大幅提升开发经验
- 架构设计能力
- 助你成为项目负责人

---

### 4. 智能面试刷题平台

**项目特点**：真实企业级项目

**核心技术**：

- **Elasticsearch 搜索**

- **Druid 并发**

- **Sa-Token 权限控制**

- **反爬虫设计**

### 5. 代码生成器共享平台

**项目特点**：深入业务场景，轮子类项目

**核心技术**：

**项目特点**：基于 Spring Boot + React 的智能化数据建表和模拟数据生成工具

## 学习建议

### 1. 循序渐进
- 不要跳跃式学习
- 从简单到复杂
- 巩固基础再进阶

### 2. 动手实践
- 不要只看不做
- 每个项目都要完整实现
- 遇到问题要自己解决

### 3. 总结输出
- 写技术博客
- 做项目笔记
- 整理知识体系

### 4. 关注新技术
- **AI 应用开发**（2025年最热门）
- 微服务架构
- 高并发系统设计
- 云原生技术

### 5. 简历准备
- 阶段1-2的项目：体现开发能力
- 阶段3的项目：体现技术广度
- 阶段4的项目：体现架构能力

## 总结

选择适合自己当前水平的项目，循序渐进地学习，注重实践和总结。2025年特别要关注 **AI 应用开发**，这是未来程序员的核心竞争力！

**记住**：项目不在多，而在于精。每个项目都要做透，能够独立完成类似的项目开发。

祝你学习顺利，早日成为优秀的工程师！💪
