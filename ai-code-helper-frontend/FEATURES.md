# AI 编程小助手 - 功能特性

## ✨ 核心功能

### 🤖 智能对话
- **实时流式响应**：AI 回答逐字显示，类似打字效果
- **上下文记忆**：同一会话中保持对话连贯性
- **编程专家**：专注于编程学习和面试辅导

### 💬 聊天界面
- **现代化设计**：IntelliJ IDE 风格的深色主题
- **清晰布局**：用户消息在右，AI 消息在左
- **实时反馈**：加载动画、消息状态提示
- **自动滚动**：新消息自动滚动到可见区域

### 🌍 多语言支持
- **双语界面**：中文 / English
- **一键切换**：右上角语言选择器
- **智能记忆**：自动保存语言偏好
- **默认英文**：符合国际化需求

### 🎨 用户体验
- **快捷键支持**：
  - `Enter` - 发送消息
  - `Shift + Enter` - 换行
- **响应式设计**：自适应各种屏幕尺寸
- **平滑动画**：消息淡入、按钮交互效果
- **状态指示**：加载中、发送中、错误提示

## 🛠️ 技术特点

### 前端技术栈
```
Vue 3 - 组合式 API
├── Vite - 快速构建工具
├── Axios - HTTP 客户端
└── EventSource - SSE 实时通信
```

### 架构优势
- **组件化开发**：模块化、可维护
- **响应式设计**：Vue 3 Composition API
- **类型安全**：良好的代码结构
- **开发体验**：热模块替换（HMR）

### 通信机制
- **SSE（Server-Sent Events）**
  - 单向实时通信
  - 自动重连机制
  - 低延迟响应
  - 浏览器原生支持

## 🔒 安全性

### 数据安全
- **本地存储**：仅保存语言偏好和会话 ID
- **无敏感数据**：不存储聊天内容
- **HTTPS**：支持加密通信

### 隐私保护
- **会话隔离**：每个标签页独立会话
- **无追踪**：不收集用户数据
- **开源透明**：代码完全开放

## 💡 使用技巧

### 提问技巧
1. **具体明确**："如何在 React 中使用 useState？"
2. **提供上下文**："我正在学习 Vue3，如何..."
3. **分步骤**：复杂问题拆分成小问题

### 最佳实践
1. **保持会话**：同一主题在一个会话中讨论
2. **及时反馈**：如果答案不满意，继续追问
3. **善用快捷键**：提高输入效率
